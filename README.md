# Hash Game

### Simple hash game with ReactJS & NodeJS & JavaScript ES6 & HTML5 & CSS3

## Install

```
    cd ~/your-directory
    git clone https://github.com/aridiosilva/reacthashgame.git ./
    npm install
```

## Execute

To get started developing right away:

start the development server with `npm start`
